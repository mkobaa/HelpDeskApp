# Help Desk Ticketing System

A simple and modern Help Desk Ticketing System built to manage support requests, track issues, and streamline communication between users and technicians.

The Help Desk Ticketing System is a web application designed to simplify the process of managing technical support requests inside an organization.

Users can create tickets to report problems or request assistance, follow the status of their requests, and communicate with support staff. Technicians can manage assigned tickets, update statuses, and resolve issues efficiently. Administrators can oversee the system, manage users, and monitor activity.

This project is built with a focus on clean architecture, separation of concerns, and practical implementation of common backend and frontend concepts.
